# PyCalculator
A simple Python calculator created for learning and practice purposes

## Features
- Basic arithmetic operations (+, -, *, /)
- Exponentiation (**) and floor division (//) 
- Minimal error handling (division by zero)
- Calculation history (When exiting the program)

## Installation & Usage
```bash
# 1. Clone the repository
git clone https://github.com/vegacy339/py-calculator.git

# 2. Run the calculator
python py-calculator/main.py
```
